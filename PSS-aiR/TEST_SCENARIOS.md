# Сценарии проверки CRUD-интерфейса PSS-aiR

## Предусловия

1. Убить процесс PSS-сервера (`AplNetTransportServTCP.exe`) если запущен
2. Запустить PSS-сервер: `"C:\Program Files (x86)\PSS_MUI\AplNetTransportServTCP.exe" /p:7239`
3. Дождаться доступности `http://localhost:7239/rest/dblist`
4. PSS-aiR запущен на порту 5002
5. Подключить PSS-aiR к БД `pss_moma_08_07_2025` (user: Administrator)

## Запуск

```bash
python PSS-aiR/test_ui_scenarios.py
```

Скрипт автоматически выполняет все предусловия.
Результаты: `PSS-aiR/test_results.html` + `PSS-aiR/test_video_pass.webm` (при 100% PASS).

## Сценарии

| # | Описание | Критерий прохождения |
|---|----------|---------------------|
| T01 | Загрузка приложения | `typeof CrudManager !== 'undefined'` = true |
| T02 | Подключение к БД | `.status-dot` имеет класс `connected` |
| T03 | Загрузка дерева папок | Есть хотя бы один `.tree-node-row` |
| T04 | Выбор папки — CRUD-тулбар | `#crudToolbar` содержит кнопки `+ Изделие`, `+ Папка` |
| T05 | Создание изделия | Изделие появляется в таблице содержимого папки |
| T06 | Редактирование изделия | Наименование обновлено в таблице |
| T07 | Удаление изделия | Изделие исчезло из таблицы |
| T08 | Создание сборки и компонента | Оба изделия видны в таблице |
| T09 | BOM — добавить существующий компонент | Компонент появляется в дереве BOM |
| T09b | BOM — создать новое изделие в составе | Новое изделие создано и видно в BOM |
| T10 | BOM — редактировать связь | Количество обновлено |
| T11 | BOM — удалить связь | Компонент исчез из BOM |
| T12 | Создание бизнес-процесса | Процесс создан (toast успеха) |
| T13 | PropertyPanel — кнопка Редактировать | При выборе изделия видна `#propEditBtn` |
| T14 | Добавление характеристики | Значение в таблице характеристик |
| T15 | Удаление характеристики | Значение исчезло |
| T16 | Добавление ресурса (для процесса) | Ресурс в таблице ресурсов |
| T17 | Удаление ресурса | Ресурс исчез |
| T18 | Привязка документа | Документ в таблице документов |
| T19 | Отвязка документа | Документ исчез |
| T20 | Очистка тестовых данных | Все тестовые объекты удалены |
| T21 | Открытие раздела 'Справочники' | Интерфейс справочников становится видимым |
| T22 | Просмотр классификаторов | Дерево классификаторов загружается (узлы или пустое состояние) |
| T23 | Навигация по дереву классификатора | Возможность раскрыть узел или выбрать узел |
| T24 | Просмотр единиц измерения | Таблица единиц загружается (таблица или empty-state) |
| T25 | Создание единицы измерения | Модалка создания, единица появляется в таблице |
| T26 | Редактирование единицы измерения | Модалка редактирования, наименование обновляется |
| T27 | Удаление единицы измерения | Диалог подтверждения, единица удаляется из таблицы |
